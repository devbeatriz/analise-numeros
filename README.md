## 💻 Projeto

Programa que analisa números entre 1 e 100 e retorna com as infomações do maior, menor, soma e média dos valores inseridos.

## 📝 Aprendizados

- [x] Uso de `Array[]`
- [x] Declaração `return`
- [x] Retornar resultado `true` e `false`
- [x] Verificar se um item faz parte de um lista
- [x] Método `push`
- [x] Método `focus()`
